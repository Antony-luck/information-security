National Information Security Competition Prototype

面向短视频多模态融合的内容安全智能研判系统

当前流程固定为“先抓取抖音链接，再自动下载视频并执行抽帧、OCR、ASR， 然后把采集结果回填到分析表单”。评论区不再做随机截取，而是先扩大候选集， 再按互动强度、回复链、作者参与和风险关键词筛出重要评论。

打开数据流检查页 打开 API 文档

分析输入

先抓取并回填项目栏，再按需要手工补充字段后启动分析。

推荐把离线模型下载到 models/asr/tiny，页面和 README 都会按这个目录优先查找。

结构化评论详情会在右侧“采集页”展示，文本框保留给你做手工补录和微调。

采集概览

显示最近一次抓取或当前表单中的核心素材信息，方便分析者快速确认对象。

尚未抓取链接数据，也尚未填充分析样例。

模块导航

点击总览页、采集页或任一模块，右侧会切换到对应独立详情页面。

详情页面

分析完成后，这里会显示总览页、结构化评论、模块证据链和下一步建议。
